Detailed Description

Class for representing smart pointer to use with pimpl idiom.

PimplPtr has the only field - typedef Type. Class created to gain 'template typedef' functionality. To use pimpl pointer you should use construction like: PimplPtr<YOUR_TYPE>::Type This is done to separate implementation of pimpl pointer (now this is boost::scoped_ptr) from usage.
