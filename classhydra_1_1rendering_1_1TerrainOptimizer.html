Detailed Description

Class for optimizing terrain surface.

It is capable for optimizing the landscape with specified maximum error value. TerrainOptimizer may be used for generating levels of details for landscape. Current implementation uses quad tree to optimize landscape. Triangulation generates triangle list wich may be optimized using hydra::rendering::MeshOptimizer.
