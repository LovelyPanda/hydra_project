Detailed Description

Sound track loader implementation using libvorbis.

Sound track loader which interprets and decodes data using libvorbis. It is cappable for loading ogg format data.

Warning:
You shouldn't create this object by hand but use special factories to get instance. You shouldn't create several instances and init them. If you initialize few such objects results depends on libvorbis.
