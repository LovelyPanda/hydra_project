Detailed Description

interface for loader to RAM or VRAM entity

This is an abstract interface for entity which is responsible for loading terrain data to RAM or to VRAM. Manager does not know anything about those things. So you should implement your own load strategy. You should implement at least 2 loaders: # From external source (hard drive, network, ...) to RAM # From RAM to VRAM

Those loaders must be passed as arguments to constructor of manager.
