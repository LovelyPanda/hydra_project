Detailed Description

Contains common classes, functions and templates which may be used by any other parts.

Namespace contains different generic classes like shared smart-pointer (SharedPtr) or implementation of factory pattern (Factory).

Now this part of Hydra project is fully in headers. So no library is built. However this can be changed in future and we would have one more library in that case.
