Detailed Description

Class for representing a cache pool (pool for storing loaded objects)

CachePool stores different objects of same type which were loaded using this library. CachePool object is used by Cache. Objects have their ids. If needed object is present in CachePool it won't be loaded second time but will be taken from cache. Cache stores copyes of loaded data as it may be changed after loading by client's code.
