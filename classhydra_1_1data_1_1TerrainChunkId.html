Detailed Description

Class for representing Chunk identificator.

Objects of this class are used to find out which Chunk object is needed. They are used for example at the loading process, when you should specify which chunk do you want to load. It contains TerrainFragmentId instance as TerrainChunk is a part of TerrainFragment.
