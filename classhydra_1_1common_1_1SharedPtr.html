Detailed Description

Class for representing shared smart pointer.

SharedPtr has the only field - typedef Type. Class created to gain 'template typedef' functionality. To use shared pointer you should use construction like: SharedPtr<YOUR_TYPE>::Type This is done to separate implementation of shared pointer (now this is boost::shared_ptr) from usage.
