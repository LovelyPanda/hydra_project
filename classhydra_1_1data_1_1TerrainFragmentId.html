Detailed Description

Identificator for TerrainFragment object.

This identificator identifies concrete TerrainFragment object from terrain surface. It stores x and y coordinate of this fragment. Do not confuse this id with TerrainChunkId which full identificator for TerrainChunk object (not a fragment).
