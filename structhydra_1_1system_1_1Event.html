Detailed Description

Struct to store window event data.

May contain mouse motion event data, key press data and so on. Also contains some enums to interpret data encoded in this structure. It is based on SDL code.

enum hydra::system::Event::KeyCode

Codes for keyboard and mouse buttons.

Most of the key codes are ASCII-mapped, so instead of Event::KEY_a you may use 'a'.

WORLD_x - are international virtual keycodes.
