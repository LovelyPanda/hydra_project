Detailed Description

Class represents generic mesh. Simple container for polygons.

Mesh contains vertices and indices to link vertices. It also contains primitive assemble mode to understand how indices are used. It may contain pointer to material object. It also may contain data which is need to create skeleton-based animation: indices of bones (joints) for each vertex and respective weights.
