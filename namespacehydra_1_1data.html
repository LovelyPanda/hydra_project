Detailed Description

Contains classes which represent different entities of data storing and aggregation.

This namespace contains such classes as Image, Model, Material, etc. Classes of this namespace contains different kinds of data and represent different entites which are containers and aggregates of data. However those classes are not common-purpose templates as such templates should lie in common namespace.
