Detailed Description

Singleton template class. Implementation of singleton design pattern.

Users can instantiate Singleton template with their classes to use them as singletons. While user's class is used via Singleton interface there will be only 1 instance of that class. And you always can get instance via instance() static method. See singleton design pattern for more info. Implementation also uses lazy initialization pattern.

Warning:
: do not forget to define static variable Singleton<T>::sInstance upon new instantiation of template.
