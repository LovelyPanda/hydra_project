Detailed Description

Factory class. Implementation of factory design pattern.

BASE argument is the base class (may be abstract). Users can instantiate Factory template to get factories for different base classes and then register some derived classes (different at a time) at appropriate factory to have ability to get objects of derived classes available by pointer of base class type. See factory design pattern for more info.
