Detailed Description

Class for representing quad tree (data structure).

QuadTree is used for generating levels of details and optimization. hydra::rendering::TerrainOptimizer uses it. ALso hydra::rendering::TerrainPreprocessor uses it. Each node contains 4 links to its childs. In this implementation it also contains link to parent, and 4 links to neighbours. User may store some additional data expanding template. Implementation does not use pointers and dynamic memory but it uses std::vector and indices. Maximum quad tree resolution is 8 levels. Minimum resolution is 1.

QuadTree may be built for square matrix of size (2^n + 1).
